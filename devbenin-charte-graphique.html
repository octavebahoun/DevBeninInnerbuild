09 — Règles d'Usage
Faire & Ne Pas Faire
→ règles non-négociables pour la cohérence de la charte
Couleurs
✓ Autorisé
Vert #22C55E sur fond dark #0A1A0D
Orange #F97316 pour CTA + accents UI
Vert clair #6EE7A0 pour textes secondaires
Toggle dark/light disponible sur tout le site
✓ Autorisé
Dot grid vert à 4% sur sections
Circuit organique dans le footer
Neurones interactifs dans le hero
✗ Interdit
Utiliser orange et vert à parts égales (ratio 50/50)
Utiliser l'orange comme couleur dominante
Fond light hors palette (utiliser #F8FAFC pour le fond et #FFFFFF pour les cards)
Superposer les 3 motifs sur la même section
✗ Interdit
Changer les couleurs du logo existant
Utiliser une couleur hors palette non validée
Opacité des motifs supérieure à 7%
Typographie
✓ Autorisé
Black Ops One pour H1, H2, H3, CTA et accroches
Syne 400–700 pour body, nav, labels
Mélanger les deux polices dans un même composant
✗ Interdit
Black Ops One sur des paragraphes ou textes longs
Police tierce non validée (Inter, Roboto, Arial…)
Black Ops One en dessous de 11px (illisible)
Illustrations & Photos
✓ Autorisé
Photo réelle pour les profils membres
Mockup / screenshot 16:9 pour les projets
Illustration vectorielle tech pour les articles
Avatar initiales vert si pas de photo
✗ Interdit
Photos stock génériques pour les articles
Images sans rapport avec la palette verte
Captures d'écran floues ou basse résolution
Visages réels non consentis dans les illustrations
Animations
✓ Autorisé
Reveal-left / right alternés au scroll
Pulse sur CTA (discret, max 2s cycle)
Neurones interactifs au hover/scroll dans le hero
✗ Interdit
Animations de couleur flash ou clignotement
Scroll reveal sur chaque élément (surcharge)
Durée d'animation >800ms (ressenti lent)
